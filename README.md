* Description de l'application :  
 * Application Carte de Visite

Cette application Android est développée avec Jetpack Compose et présente une carte de visite numérique simple et élégante.
Elle affiche une photo de profil, le nom, la profession ainsi que les informations de contact (téléphone, réseaux sociaux et email) dans une interface moderne.

 * Fonctionnalités

Interface utilisateur moderne avec Jetpack Compose

Affichage d’une carte de visite centrée

Icônes de contact (téléphone, partage, email)

Mise en page claire avec Column et Row

Thème personnalisé et couleurs harmonieuses

 * Technologies utilisées

Kotlin

Jetpack Compose

Android Studio
